# Verkehslage im Pustertal

Dieses Projekt dient der automatisierten Auswertung und Visualisierung von Verkehrsdaten für die Zählstelle Vintl an der Staatsstraße SS49. Ziel ist es, saisonale Schwankungen des Verkehrsaufkommens im Jahresverlauf sichtbar zu machen.
